A wedding planner website. Made with ❤ by Sparsh Kumar.

This website is based on Microservices Architecture. It consist of 5 microservices - Services, Orders, Pages and Accounts, all written using Django, and Email Notification, written using Springboot. Django Templates are used for Dynamic rendering of HTML and Javascript codes. Bootstrap is used for responsive frontend. All the microservices use RabbitMQ and Celery for asynchronous intercomponent communication. All the microservices are made as Docker containers and hosted on AWS EC2. A nginx server is used as Reverse Proxy. All the static and media files are stored on AWS S3. Payments are handled using Razorpay Payment Gateway. Finally DuckDns and Let's Encrypt are used for Dynamic DNS and HTTPS SSL certificate.
